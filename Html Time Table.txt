3.Html attributes -
  Attributes are always specified in the start tag
  Attributes usually come in name/value pairs like: name="value" 
  1-----The <a> tag defines a hyperlink. The href attribute specifies the URL of the page
   <!DOCTYPE html>
   <html>
   <body>
   <h2>The href Attribute</h2>
   <a href="https://www.w3schools.com">Visit W3Schools</a>
   </body>
   </html>
   2-----The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed.
   <!DOCTYPE html>
   <html>
   <body>
   <h2>The src Attribute</h2>
   <img src="img_girl.jpg" width="500" height="600">
   </body>
   </html>
   There are two ways to specify the URL in the src attribute:1.Absolute url - not used because of some situations
                                                              2.Relative url - used mostly
   3.------The style attribute
    The style attribute is used to add styles to an element, such as color, font, size, and more.
     <!DOCTYPE html>
     <html>
     <body>
     <h2>The style Attribute</h2>
     <p>The style attribute is used to add styles to an element, such as color:</p>
     <p style="color:red;">This is a red paragraph.</p>
     </body>
     </html>
